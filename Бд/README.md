# Подключение бд к компьютеру

## Шаг 2: Подключить переносной диск и скопировать файл
1.	Вставьте диск F: в новый компьютер
2.	Убедитесь, что файл Tur_Comp_backup.sql находится на диске

## Шаг 3: Создать пустую базу данных
Откройте командную строку (CMD) или PowerShell от имени администратора и выполните:

cd "C:\Program Files\PostgreSQL\17\bin"
.\createdb -U postgres Tur_Comp

Вас спросят пароль — введите пароль пользователя postgres. 

## Шаг 4: Восстановить базу из дампа
.\psql -U postgres -d Tur_Comp -f F:\Tur_Comp_backup.sql
Снова введите пароль. Процесс восстановления может занять некоторое время в зависимости от размера базы данных. 

## Шаг 5: Проверить результат
Подключитесь к базе и посмотрите список таблиц:
.\psql -U postgres -d Tur_Comp
После подключения выполните: 
\dt
Это покажет список всех таблиц в базе. Если таблицы отображаются — перенос прошел успешно.
Выйти из psql можно командой:
\q
